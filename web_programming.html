Materi Pemrograman Web

1. HTML

HTML digunakan untuk membuat struktur dasar halaman web.

<!DOCTYPE html>
<html>
    <head>
        <title>Halaman HTML</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

2. CSS

CSS digunakan untuk mengatur tampilan halaman web.

body {
    background-color: lightblue;
    text-align: center;
}

3. JavaScript

JavaScript digunakan untuk menambahkan interaksi pada halaman web.

<script>
    alert("Selamat datang di website saya!");
</script>

4. PHP

PHP adalah bahasa pemrograman server-side untuk membuat website dinamis.

<?php
    echo "Halo, ini adalah kode PHP!";
?>
